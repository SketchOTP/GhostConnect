# Privacy Policy — GhostConnect

GhostConnect is a local-first health dashboard application that connects to Fitbit after you authorize access.

## Data Collected
- Fitbit data you authorize (activity, sleep, heart rate, etc.)
- Local configuration required to operate the application

## Data Usage
- Data is stored locally on your device.
- Data is used to generate health dashboards and summaries.
- If AI assessment is enabled, summarized daily metrics may be sent to the OpenAI API for analysis.

## Data Storage
- All Fitbit data is stored locally in a local database.
- OAuth tokens are stored locally and should be encrypted at rest.

## Data Sharing
GhostConnect does not sell or share your data.
Data is only transmitted to:
- Fitbit (to retrieve authorized data)
- OpenAI (if AI assessment is enabled)

## Data Deletion
You may delete all data by:
- Deleting the local database file
- Revoking access in your Fitbit account settings

## Contact
For support or questions, open an issue on the GitHub repository.
